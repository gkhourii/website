# Nextra Blog Portfolio

This is a personal portfolio and blog built with [Next.js](https://nextjs.org), [Nextra](https://nextra.site), and TypeScript.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Open [http://localhost:3000](http://localhost:3000)** with your browser to see the result.

## Content

- **Home:** `app/page.mdx`
- **Blog Posts:** Add `.mdx` files in `app/posts/`.
  - Example: `app/posts/hello-world.mdx`
- **Configuration:** Update `theme.config.jsx` and `app/layout.tsx`.

## Deployment

This project is ready to be deployed on [Vercel](https://vercel.com/new).

1.  Push your code to a Git repository (GitHub, GitLab, Bitbucket).
2.  Import the project in Vercel.
3.  Vercel will automatically detect Next.js and deploy.
